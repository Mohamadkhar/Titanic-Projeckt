# Notebooks

Dieser Ordner enthält die Jupyter Notebooks für die Datenanalyse.

## Notebook-Reihenfolge

Die Notebooks sollten in folgender Reihenfolge durchgeführt werden:

1. **01-data-exploration.ipynb** - Erste Datenexploration und Übersicht
2. **02-data-cleaning.ipynb** - Datenbereinigung und Preprocessing
3. **03-analysis.ipynb** - Statistische Analysen und Hypothesentests
4. **04-visualization.ipynb** - Erweiterte Visualisierungen und Ergebnisse

## Weitere Lab-Notebooks

Die Notebooks aus dem Lab-Kurs (02-jupyter.ipynb bis 17-pytorch.ipynb) befinden sich im Hauptverzeichnis und dienen als Referenz.

## Best Practices

- Nutze aussagekräftige Markdown-Zellen zur Dokumentation
- Kommentiere deinen Code
- Erstelle reproduzierbare Analysen
- Speichere aufbereitete Daten in `../data/processed/`
- Nutze Funktionen aus `../src/utils.py` für wiederverwendbaren Code
